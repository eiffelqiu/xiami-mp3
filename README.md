xiami-mp3
=========
How to get xiami website's mp3 download address

	http://xiami-mp3.herokuapp.com/

Usage
=======

1: prepare for rubygems
-------
	$ bundle install

2: how to run?
-------
	$ ruby xiami.rb

![usage1](https://raw.github.com/eiffelqiu/xiami-mp3/master/capture1.png)

3: get the mp3
-------
	$ open http://localhost:4567
